# Async/Await http-requests

Estos ejemplos cubren el **90%** de lo que usarás día a día con `async/await`. La clave es:

- **Siempre usar `try/catch`** para manejar errores.
- **Usar `Promise.all()`** cuando las operaciones son independientes.
- **Hacer secuencial con `await`** cuando una operación depende de la anterior.
